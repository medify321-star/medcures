## 📞 Important Notes

### Keep Terminal Open
- If you close the terminal with PM2, backend stops
- That's OK! It will just be local only
- When you deploy to Google Cloud, it doesn't matter

### Don't Commit .env Files
- `.gitignore` already protects it
- Your secrets stay safe ✅
- Never share `.env` files!

### Update Your Code Later
```powershell
# Make changes locally
# Test on http://localhost:3001
```
